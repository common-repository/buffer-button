== Description ==

Allows you to add the Buffer Button to your blog. [Read more](http://blog.bufferapp.com/say-hi-to-the-buffer-button-for-blogs).

== Installation ==

You should be able to easily install this straight from the plugins directly by searching for "Buffer". To install manually:

1. Upload `buffer-button.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
